## File Structure
Within the download you'll find the following directories and files:

```
material-kit
  ├── assets
  │   ├── css/
  │   ├── fonts/
  │   ├── img/
  │   ├── js
  │   │   ├── core
  │   │   ├── plugins
  │   │   └── material-kit.min.js
  │   └── scss
  │       ├── material-kit/
  │       └── material-kit.scss
  ├── pages/
  ├── sections/
  ├── CHANGELOG.md
  ├── gulpfile.js
  ├── package.json
  ├── presentation.html
```
